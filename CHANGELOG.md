# Changelog

All notable changes to this project will be documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and the project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-11-11

### Added
- Initial public release of the P2P client component under the EUPL-1.2 licence.
- Shared configuration DTOs and providers for accessing hub settings.
- Composite validation pipeline with built-in rules for endpoints, labels, and metadata.
- Composite metadata providers and normaliser for peer registration payloads.
- Reusable client and query services for registering peers and discovering remote nodes.

### Changed
- Composer metadata updated for public distribution (author, keywords).
